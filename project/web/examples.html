spice3d example circuits

Each link below opens the Godot simulator with the named example preloaded. The simulator starts paused until the PDK is fetched (first visit per PDK family downloads ~20 MB into IndexedDB; cached afterwards).

button sky130 1.8 V 1 button One clickable button drives the input of a sky130_fd_sc_hd__inv_1 standard-cell inverter; the output net btn_out_n visibly toggles in response. Minimal end-to-end smoke test of the whole pipeline. open →
ro sky130 1.8 V none Five-stage ring oscillator built from sky130_fd_sc_hd inverters. No inputs — wires light up in a rolling wave as the loop self-oscillates. Stresses the continuous-step-N sim path: nothing to click, just steady-state behavior. open →
gf180mcu_nand2 gf180mcu 5.0 V 2 buttons (A1, A2) Transistor-level 2-input NAND built from nfet_05v0 and pfet_05v0 primitives (no standard-cell wrapper). Two clickable buttons drive the A1 and A2 inputs; the output net ZN follows the NAND truth table. Adapted from sifferman/gf180mcu_schematics. open →
3bit_counter_busses sky130 1.8 V 1 button (CLK pulse) 3-bit counter built from three sky130_fd_sc_hd__dfxtp_1 flip-flops (vector-instantiated as xdff[2:0]) feeding a 3bit_incrementor subcircuit. Each click of the CLK button advances the counter; the state_q[2:0] and state_d[2:0] bus wires light up with one of 8 distinct hues per integer value. First example exercising bus wires (xschem name[MSB:LSB] notation, bit aggregation, and the per-state color palette). open →

How to add an example

Drop a self-contained .sch (+ any custom .sym files) into project/examples/<name>/, register the directory in KNOWN_EXAMPLES_BY_DIRECTORY_NAME inside project/main.gd, then add a row to this table. The pdk_family field selects which PDK gets fetched — current options are "sky130" and "gf180mcu".
